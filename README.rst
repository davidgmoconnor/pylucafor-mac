Python API for the `Luxafor Flag <https://luxafor.com/luxafor-flag/>` on macOS


Installation
------------

Clone this repo::

    $ git clone https://github.com/davidgmoconnor/pyluxafor-mac

A PyPi installation via pip will follow soon.
